🌐 Partie 1 — Site Web sur Vercel
Déployer les 35 pages HTML de NutriCam gratuitement en moins de 30 minutes.
1
Créer un compte GitHub (si pas encore fait)
⏱ 2 min

GitHub servira à stocker votre code. Vercel se connecte à GitHub pour déployer automatiquement.

🔗
Allez sur github.com → cliquez "Sign up" → entrez votre email pitchouduplece9@gmail.com

Choisissez le plan Free (gratuit pour toujours pour les projets publics).

2
Créer un dépôt GitHub "nutricam"
⏱ 3 min
Terminal / Git Bash
# Sur votre ordinateur, créez un dossier et initialisez Git
mkdir nutricam-website
cd nutricam-website
git init

# Copiez TOUS vos fichiers nutricam-*.html dans ce dossier
# Puis créez un fichier index.html qui redirige vers le hub
echo '<meta http-equiv="refresh" content="0;url=nutricam-hub.html">' > index.html

# Ajoutez tout et faites le premier commit
git add .
git commit -m "🚀 NutriCam v1.0 - Initial deploy"

# Connectez à GitHub (remplacez YOUR_USERNAME par votre pseudo)
git remote add origin https://github.com/YOUR_USERNAME/nutricam.git
git branch -M main
git push -u origin main
💡
Si vous n'avez pas Git installé : téléchargez-le ici (Windows) ou utilisez directement l'interface GitHub pour uploader les fichiers.
3
Déployer sur Vercel (gratuit)
⏱ 5 min

Vercel héberge des sites web gratuitement avec HTTPS automatique et CDN mondial.

Allez sur vercel.com → "Sign up with GitHub"
Cliquez "Add New Project" → Sélectionnez votre repo "nutricam"
Framework Preset: sélectionnez "Other" (pas Next.js, ce sont des HTML simples)
Cliquez "Deploy" → Attendez 60 secondes
✅ Votre site est en ligne sur nutricam.vercel.app !
🎉
Chaque fois que vous faites un git push, Vercel redéploie automatiquement en moins de 30 secondes !
4
Connecter le domaine nutricam.cm
⏱ 30 min (propagation DNS)

Option A — Domaine camerounais (.cm) : Achetez via un registrar local.

🌍
Registrar camerounais recommandés :
• afriregister.cm — registrar officiel Cameroun
• hostingsolutions.cm
Prix estimé : 15 000–25 000 FCFA/an pour un .cm

Option B — Domaine international (.com) : Plus simple, accessible partout.

Configuration DNS Vercel
# Dans Vercel → Project Settings → Domains
# Ajoutez votre domaine : nutricam.cm (ou nutricam.com)

# Vercel vous donnera 2 enregistrements DNS à configurer :
Type: A     Name: @      Value: 76.76.19.61
Type: CNAME  Name: www    Value: cname.vercel-dns.com

# Entrez ces valeurs dans le panneau DNS de votre registrar
# La propagation prend 5 min à 48h selon le registrar
📱 Partie 2 — Application Mobile
Publier l'app NutriCam sur App Store (iOS) et Google Play Store (Android).
5
Installer Node.js et Expo CLI
⏱ 10 min
💻
Téléchargez Node.js 18 LTS (nécessaire pour Expo).
Terminal
# Installer Expo CLI globalement
npm install -g expo-cli eas-cli

# Aller dans le dossier de l'app mobile NutriCam
cd nutricam-mobile

# Installer les dépendances
npm install

# Tester en local (scan QR code avec Expo Go sur votre téléphone)
npx expo start

# ✅ Scannez le QR code avec l'app "Expo Go" (iOS/Android)
# → Vous verrez NutriCam tourner sur votre téléphone !
6
Créer les comptes développeur

25 $ une seule fois (~15 000 FCFA)

→ S'inscrire Google Play
💡
Conseil : Commencez par Google Play (moins cher, plus rapide). Votre compte Google : pitchouduplece9@gmail.com
7
Build et soumission avec EAS (Expo Application Services)
⏱ 20 min (build ~15 min)
Terminal
# Se connecter à votre compte Expo
eas login
# Email: pitchouduplece9@gmail.com

# Configurer le projet (une seule fois)
eas build:configure

# ── BUILD ANDROID (APK pour Google Play) ──
eas build --platform android --profile production
# ⏳ Attend ~15 min sur les serveurs Expo
# ✅ Télécharge l'APK/AAB automatiquement

# ── BUILD iOS (Archive pour App Store) ──
eas build --platform ios --profile production
# Nécessite le compte Apple Developer

# ── SOUMISSION DIRECTE aux stores ──
eas submit --platform android
eas submit --platform ios
🎉
EAS gère automatiquement les certificats de signature iOS et Android. Vous n'avez pas besoin de Xcode ou Android Studio !
