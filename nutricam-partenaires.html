📡 API NutriCam

Intégrez notre base nutritionnelle dans votre app

L'API NutriCam expose nos 200+ recettes camerounaises, valeurs nutritionnelles et recommandations IA. Idéale pour les développeurs d'applications santé et nutrition.

🍲
Base recettes complète : 200+ plats avec ingrédients, étapes, valeurs nutritionnelles par portion
🔍
Recherche intelligente : Full-text multilingue, filtres par région, catégorie, calories, profil
📦
Scanner produits : Lookup par code-barres, 1200+ produits camerounais

Génération IA : Endpoint plan repas personnalisé selon profil utilisateur
